# SOC Monitoring Dashboard

An interactive **Security Operations Center (SOC) dashboard** built using **Next.js and TailwindCSS** to simulate real-world security monitoring environments.

This project demonstrates core SOC analyst concepts such as:

- Threat monitoring
- SIEM log analysis
- Cyber attack visualization
- Incident investigation
- Threat intelligence

# Features

### Threat Detection Chart
Visualizes security events and threat trends in real-time.

### Live SIEM Logs
Simulated security logs similar to those generated by tools like **Splunk / ELK**.

### Cyber Attack Map
Global attack visualization showing simulated threat activity.

### Investigation Simulator
SOC workflow simulation for analyzing security alerts.

### Threat Intelligence Feed
Displays simulated threat intelligence updates.

### Incident Timeline
Tracks security incidents and investigation events.

### SOC Terminal
Terminal-style interface showing security operations activity.

### Attacker IP Intelligence
Displays attacker IP addresses, geolocation, and threat types.

### Splunk Query Simulator
Demonstrates how SOC analysts query security logs.

### Threat Intelligence Score
Risk scoring engine showing current threat level.
